## HOW TO RUN

- Dillinger requires Node.js v18+ to run.
- Install the dependencies and devDependencies and start the server.

```
$npm i
$npm start

$npm run watchDB
- To view db.json
$node src/api/video.js
- To view and render video and image

Open on web: http://localhost:1235
```
